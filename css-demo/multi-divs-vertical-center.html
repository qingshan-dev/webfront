    - know child(父元素宽高未知，子元素宽高已知)
      - 绝对定位法 + margin 负值
    - Unknown Child（未知宽高度的父级元素，未知宽高的子级元素，要子级元素垂直居中?）
      - 利用 table 的属性来进行设置
      - 添加 ghost 元素（或者利用伪类构造）,利用 vertical-align 需要参照物的原理吗？
         set up a "ghost" element inside the parent that is 100% height, then we vertical-align: middle both that and the element to be centered, we get the same effect.

         Note: The 0.25em nudge-back is a little janky. To do it perfectly, you could set font-size: 0; on the parent and then notch the font size back up inside the content container.
      - 应用该方法，父级元素必须有指定的高度，height，而不能仅有min-height;
    - 本质上还是回到了块状元素和内联元素的区别。
